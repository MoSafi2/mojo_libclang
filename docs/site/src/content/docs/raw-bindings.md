# Raw Bindings

The low-level FFI boundary is generated by `mojo-bindgen` and patched for local
libclang ABI behavior.

The raw generated module is `clang._ffi`. High-level wrappers live under
`clang/` and are re-exported from `clang.cindex` for user code.

The canonical implementation notes for ABI fixes, shim handling, generated
files, and current generator limitations live in
[`raw_bindings.md`](https://github.com/MoSafi2/mojo_libclang/blob/master/raw_bindings.md).
